予約の前にご確認ください

お申し込み前に、知っておいていただきたいことをまとめました。

  1. 締切まで: このサイトからの新規お申し込み・内容の変更・キャンセルができます。締切の日時は開催日ごとに異なり、画面に表示される締切の日時が基準です。多くの場合、開催の2日前 15時（日本時間）が目安です。
  2. 開催人数: 締切の時点で参加チームが3チームに満たない場合は、大会が行えないことがあります。その場合は運営からご連絡します。
  3. 定員: 午前の時間帯ごとに入れるチーム数に上限があります。全体としても、グラウンドの都合でお申し込みの受付に上限があります（通常は6チーム前後。大会の設定により変わります）。
  4. 午前と午後: 午前は、お選びいただいた時間帯で対戦が決まります。午後は、締切のあと運営の手配で組み合わせます。チームの「レベル感」は参考として選んでいただくだけで、他チームと違っていてもお申し込みはできます。
  5. 学年: お子さまの学年は、その日の大会の学年帯に合うものをお選びください。複数学年がいる場合は、人数が多い方の学年に合わせてください。
  6. 参加人数と昼食: 参加人数はコーチや保護者など当日いらっしゃる方の合計です。昼食の数は、参加人数と同じでなくてもかまいません。
  7. 雨天・中止: 開催や中止のお知らせは、前日のメールなどでお送りします。天候が急変した場合は、その時点での運営の判断でお知らせすることがあります。
  8. 確認コード: お申し込みが終わると確認用のコードが表示されます。後から予約内容を見るとき・キャンセルするときに必要です。メールが届かない・コードを失くしたときは、配布資料の連絡先へお問い合わせください。
  9. 合宿・お泊まり: 日帰りのお申し込みとは別に、「合宿のご案内」からご相談を受け付けています。
当日の欠席や遅刻、試合の進め方、昼食の配り方などは、現場の運営でお願いします（このサイトでは管理しません）。

予約の確認・キャンセル

予約のときにお渡しした確認コードを入力し、「確認」を押してください。受付中で、かつ締切前まで、このサイトから参加人数・昼食の数・代表者の連絡先を変えたり、キャンセルしたりできます。

確認コードで確認できないとき

  • 前後の空白や改行が入っていないか確認し、もう一度貼り付けてください。
  • 64 文字の英数字（0–9 と a–f）だけか確認してください（コピー漏れ・1 文字多い等がよくあります）。
  • 開催日から 30 日を過ぎると確認できません（確認コードは無効になります）。
  • それでも解決しない場合は、配布資料や案内に書いてある大会運営の連絡先へお問い合わせください。このサイトから確認コードを再送することはできません。
チーム名・メールアドレス・午前の時間帯の変更は、このサイトからはできません。運営へお問い合わせください。

合宿のご案内

当サイトでは、宿泊を伴う交流試合・合宿についてもご相談いただけます。交流試合を含めたご希望や、宿泊中心のご相談も可能です。

参加人数や詳細が未確定でも、まずは希望の日程感とプランの目安からお気軽にご相談ください。お送りいただいた内容を確認のうえ、運営よりご案内します。

この時点では予約の確定ではありません。空き状況や部屋の厳密な自動突合は行わず、受付後はメール等でのやり取りを想定しています。

このサイトでお手伝いすること

  • 合宿・宿泊の相談の受付
  • 運営が受入可否を判断するための情報の共有
  • 開催前までの事前のご案内（メール等）

開催が決まった当日の試合順の細かい調整、チーム間の進行、当日運営の詳細などは、各チーム・現場での対応を前提としており、本サイトに進行管理・合宿専用の対戦表・当日運営の管理機能はありません。
