昼食（SDGs・地域連携の注記イメージ） 当日は給食・食堂の運用に合わせた案内文をここに集約（持ち込み可否・支払い方法は運営文面で固定）。

当日の流れ（例）
午前（1試合程度） → 昼食 → 午後（複数試合の場合あり）

  1. 対象学年帯は開催日ごとに決まります。フォームでは代表学年（1〜6年）を選択します（帯と整合しない学年は予約できません）。
  2. 午前は空き枠を選びます。枠が満枠の時間は選べません（実装: availability の full / bookable）。
  3. チームカテゴリは「ハイレベル（strong）」「ポテンシャル（potential）」の2択（実装と同ラベルイメージ）。
  4. 雨天・中止の連絡は、前日メール・即時メール等の運用に従います（詳細は運営告知）。
  5. 予約の変更・取消は、開催日が open かつ Web 締切前のみ可能です。締切時刻は開催日ごとに reservation_deadline_at（既定は開催2日前 15:00 JST）で表示します。
  6. 予約完了後は確認コードが必要です。メール未達時は運営窓口での案内を想定（現仕様の補足）。
