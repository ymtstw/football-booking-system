改善ポイント一覧

  1. ファーストビュー内にメインCTA配置: ヒーロー（無料開放バナー）直下に「開催日を確認する」を配置。スマホでもスクロールほぼ不要で到達可能。
  2. スティッキー下部CTA（モバイルのみ）: スクロール中も画面下部にCTAを固定表示。どの位置からでもワンタップで遷移可能。
  3. 情報カード 2×2 グリッド + アイコン化: 4枚の情報カードをスマホで縦4枚積みではなく2×2に圧縮。要点を一目で把握可能。
  4. 昼食セクションを "要確認カード" として常時表示: 重要度が高いためアコーディオンに隠さず、○×アイコン付きの視覚的な項目リストで瞬時に把握できるデザインに。箇条書きを独立カードに分割することで、1行ずつスキャンしやすい。
  5. 予約ルール・合宿相談はアコーディオン化: 初回判断に必須ではない情報だけ折りたたみ、必要な人だけ開く。
  6. 予約締切はコンパクトな警告バナー: 大きなコールアウトをやめ、1〜2行の amber バナーに集約。最も重要な情報を見落とさずに短く伝達。
  7. スマホ用ステッパーの簡素化: 4段フル表示をやめ、現在位置を強調したブレッドクラム型に。縦幅を約半分に削減。
参考: スマホで CTA 到達までに必要な「指の動き（スクロール量）」は、現状比でおよそ 1/4〜1/5 に削減される想定です（当ビューア内でのスクロールテスト）。
