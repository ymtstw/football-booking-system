予約の前にご確認ください

お申し込み前に、運営方法や予約の流れを簡単にご案内します。

  1. 予約できるのは午前の1枠です。 午後の対戦カードは、締切後に運営側で調整します。
  2. 締切までは予約内容の確認とキャンセルができます。 締切日時は開催日ごとに画面に表示されます。
  3. 開催は参加チーム数によって決まります。 一定数に満たない場合は中止となることがあります。
  4. 参加人数には選手・スタッフ・帯同者を含めてください。 昼食数は参加人数と同じでなくても大丈夫です。
  5. 予約完了後に確認コードが表示されます。 後から予約内容を見るときに必要なので、必ず保存してください。
学年やチームレベルは、できるだけ近い相手と対戦できるようにするための参考情報です。希望どおりにならない場合もあります。

合宿のご相談について
